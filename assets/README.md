# assets/ ディレクトリ

このディレクトリに以下のファイルを配置してください。

## 必要なファイル

### bgm.mp3（任意・推奨）
- 動画のバックグラウンドミュージック
- ロイヤリティフリーの音楽を使用すること
- 推奨: 穏やか・清潔感のあるBGM（天気予報らしい雰囲気）
- 長さ: 30秒〜数分（短くても自動ループします）

## BGM 素材おすすめサイト（商用利用可）

| サービス | URL | 備考 |
|---------|-----|------|
| Audiostock | https://audiostock.jp | 日本語サービス・品質高い |
| DOVA-SYNDROME | https://dova-s.jp | 完全無料・利用しやすい |
| Musescore | https://musescore.com | 楽譜も豊富 |
| Pixabay Music | https://pixabay.com/music | 英語だが使いやすい |
| Free Music Archive | https://freemusicarchive.org | CC ライセンス |

## 利用上の注意

- BGMのライセンスを必ず確認してください（YouTube収益化を行う場合は特に注意）
- ファイルは `assets/bgm.mp3` として配置してください
- bgm.mp3 が存在しない場合はBGMなしで動画が生成されます（エラーにはなりません）
